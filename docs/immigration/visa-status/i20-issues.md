# I-20问题处理

## I-20常见问题

### 问题1: I-20即将过期

**症状**:
- Program end date在3个月内
- 无法按时完成学业

**解决方案**:
1. 立即联系DSO
2. 申请I-20延期(Program Extension)
3. 需要的理由:
   - 学术原因(换专业、加double major)
   - 健康原因
   - 英语能力需要提升
   - Thesis/dissertation需要更多时间

**时间线**:
- 必须在I-20到期前申请
- 通常1-2周获得新I-20

### 问题2: I-20信息错误

**常见错误**:
- 姓名拼写错误
- 出生日期错误
- 学位级别错误
- Major错误

**立即行动**:
1. 发现立即通知DSO
2. 提供正确信息证明
3. DSO会签发corrected I-20
4. 保留旧I-20副本

⚠️ **重要**: 如果已经离境，可能需要重新签证

### 问题3: I-20丢失

**立即采取**:
1. 报告DSO
2. 申请replacement I-20
3. 通常免费或小额费用
4. 1-2周获得

**如果在美国境外丢失**:
- 联系学校国际学生办公室
- 要求email扫描件
- 可能需要等待原件才能入境

### 问题4: SEVIS ID问题

**SEVIS被terminate**:
- 原因: 身份失效、违规
- 后果: 必须离境或恢复身份
- 解决: Reinstatement申请(复杂且昂贵)

**SEVIS被transfer**:
- 转学时正常流程
- 需要在15天内到新学校报到

## I-20延期申请

### 申请材料
- [ ] I-20延期申请表(学校表格)
- [ ] Advisor推荐信
- [ ] 学术计划说明
- [ ] 成绩单
- [ ] 财力证明(可能需要)

### 理由撰写
```
由于[具体原因]，我无法在当前I-20的program end date
[日期]前完成学业。我需要延长[X]个学期/月。

[详细解释为什么需要更多时间]

我计划在[新的预计毕业日期]完成所有学位要求。
```

## Travel Signature管理

### 何时需要
- 每次离开美国前
- 签名有效期12个月
- 在I-20第三页

### 如何获取
1. 联系DSO
2. 面谈或email(看学校政策)
3. 当天或1-2天内获得
4. 出发前1-2周办理

### 不能travel的情况
- I-20即将过期(30天内)
- 成绩不好或probation
- SEVIS有问题
- 未注册下学期课程

## 紧急I-20问题

### 人在境外，I-20有问题
1. 联系学校紧急联系人
2. 说明情况
3. 可能需要:
   - 延期返校日期
   - 重新申请签证
   - 新的I-20

### 入境时I-20被质疑
- 保持冷静
- 诚实回答CBP问题
- 提供辅助文件
- 如被拒入境，联系学校律师

## 保护你的I-20

### ✅ 最佳实践
- 拍照保存所有I-20副本
- Email给自己云端备份
- 保留所有旧I-20
- 定期检查信息正确性
- 提前3个月检查到期日

### 📋 I-20检查清单
每学期开始:
- [ ] Program end date足够
- [ ] Travel signature在12个月内
- [ ] SEVIS状态active
- [ ] 个人信息正确
- [ ] 已注册足够学分
