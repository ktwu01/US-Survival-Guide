# 关于本项目

## 项目起源

这个项目诞生于作者本人差点跌入"斩杀线"的亲身经历。

两年前，一次突发的医疗紧急情况让我收到了一张近$20,000的医疗账单。作为F-1学生，当时的我几乎要被这张账单压垮。

幸运的是，在使用GPT的帮助下，我学会了如何与医院谈判，最终将账单降至$4,000，并申请到了分期付款。这段经历让我意识到：

**在美国生存，知道规则和不知道规则的人，命运完全不同。**

---

## 为什么创建这个指南？

在研究如何处理医疗账单的过程中，我发现：

1. **大型语言模型（GPT/Claude等）不愿意提供具体的维权策略**
   - 出于法律风险考虑，AI往往给出模糊的建议
   - 但现实生活中，你需要的是可执行的具体步骤

2. **中文互联网上缺乏系统化的美国生存指南**
   - 信息零散在小红书、1point3acres等平台
   - 没有一个集中的、持续更新的资源库

3. **很多人不知道自己有哪些权利**
   - 医疗账单可以谈判
   - 信用卡争议是合法权利
   - 很多"必须支付"的费用其实可以减免

**这个指南的目标，就是填补这些空白。**

- 所有内容必须可执行

- 不鼓励欺诈或不当行为

### 3. 持续更新
- 政策和规则在变化
