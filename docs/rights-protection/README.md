## 核心原则

### 1. 了解你的权利
- 租客权利（Tenant Rights）
- 工人权利（Worker Rights）
- 消费者权利（Consumer Rights）
- 学生权利（Student Rights）

### 2. 文档记录一切
- 所有沟通留痕
- 拍照/录音（合法前提下）
- 保存所有收据和合同

### 3. 升级路径
```
1. 直接沟通
    ↓
2. 书面通知
    ↓
3. 第三方调解
    ↓
4. 小额法庭/诉讼
```
