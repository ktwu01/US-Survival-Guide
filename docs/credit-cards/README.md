# 💳 信用卡维权

> 信用卡争议（Dispute/Chargeback）是合法且有效的消费者权益保护工具。本指南教你如何正确使用信用卡维权。

## 核心要点（预览）

- **信用卡争议**是法律赋予的权利，不是"恶意行为"
- **60天**是大多数情况下的争议时限
- **文档记录**是成功维权的关键
- **不要滥用**，否则可能被银行关户

---

## 快速参考

### 可以发起争议的情况

- 商品未收到
- 商品与描述不符
- 商品损坏/有缺陷
- 重复扣费
- 未经授权的扣费
- 商家拒绝退款

### 不应发起争议的情况

- 单纯后悔购买（Buyer's remorse）
- 商家已经退款
- 超过争议时限
