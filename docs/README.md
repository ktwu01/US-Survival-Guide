# 美国斩杀线生存指南

## 什么是"斩杀线"？

"斩杀线"指的是当美国普通人（尤其是留学生）的**财务储备、精神支撑与生存资源**跌破临界点后，会触发不可逆的连锁崩盘，最终被社会系统"淘汰"的生存阈值。

一旦越过斩杀线，你可能面临：
- 医疗账单摧毁信用记录
- 无力支付房租被驱逐
- 失去合法身份（F-1/OPT/H-1B断签）
- 精神崩溃无法继续学业或工作
- 陷入债务螺旋无法翻身

**本指南的目标**：帮助在美中国留学生和华人识别风险、掌握维权技巧、建立财务防御体系，避免跌入斩杀线。

---

## 为什么需要这个指南？

在美国生活，你会发现：
- **大型语言模型（GPT/Claude等）不愿意告诉你的实战技巧**
- 医疗账单可以砍价，而且可以砍很多
- 信用卡争议（dispute）是合法且有效的维权手段
- 很多"必须支付"的费用其实可以通过正当途径减免
- 知道规则和不知道规则的人，命运完全不同

**这个仓库收集的是那些"教科书不教、AI不敢说、但你必须知道"的生存知识。**

## 如何使用本指南

### 预防性阅读
如果你目前财务状况稳定，建议：
1. 阅读 `docs/emergency-finance/` 了解风险预警信号
2. 保存 `templates/` 中的模板信件以备不时之需
3. 收藏 `resources/` 中的关键联系方式

### 应急响应
如果你正面临财务危机，立即查看：
1. `guides/` 中的应急处理指南
2. `docs/` 对应类型的详细文档
3. `templates/` 中的现成模板，改好就能用
